# Python and Pandas Exercises

This repository contains Python and Pandas exercises completed during my **Data Technician Skills Bootcamp**. The notebooks demonstrate my progress from learning basic Python programming to working with structured datasets using Pandas.

## Skills Developed

* Used variables, user input, data types and arithmetic operators.
* Built programs using `if`, `elif` and `else` statements.
* Used `for` and `while` loops to repeat tasks and process values.
* Created practical programs including calculators, unit conversions, prime-number checks and factorial calculations.
* Imported CSV and Excel datasets into Pandas DataFrames.
* Inspected datasets using methods such as `head()`, `info()` and `describe()`.
* Selected, filtered, sorted and updated rows and columns.
* Converted data types and cleaned text-based values.
* Identified, counted and handled missing values.
* Used summary statistics and functions such as `mean()`, `sum()` and `value_counts()`.

These exercises strengthened my understanding of Python fundamentals, problem-solving and using Pandas to prepare, explore and analyse data.
